Нерассортированные данные
====
В этом разделе данные извлечённые из госреестров по доменам 

* budgetreg - данные извлеченные из реестра участников и неучастников бюджетного процесса
* customers - данные извлечённые из реестра госзаказчиков по 44-ФЗ и 223-ФЗ из Госзатрат (http://clearspending.ru)
